# HostelCare
Complaint & Maintenance App

A React Native (Expo) app for hostel students to submit complaints and for wardens to manage and resolve them efficiently.

 Features:
 
Student-

1. Create complaints (category, room, description, photo) <br/>
2. Track status: Submitted → In Progress → Resolved → Rejected  <br/>
3. Comment on complaints  <br/>
4. Upvote other complaints  <br/>
5. View all submitted complaints  <br/>

 Warden / Staff- 

1. View all complaints with filters (category, status, block/floor)  <br/>
2. Assign complaints to staff  <br/>
3. Update status + add resolution notes  <br/>
4. View comments and timeline  <br/>
5. Insights dashboard (category counts, top issues)  <br/>

 Tech Stack:

Frontend: React Native (Expo)  <br/>
Backend: Node.js, Express, Prisma  <br/>
Database: PostgreSQL / MongoDB  <br/>
